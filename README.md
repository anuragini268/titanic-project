# Titanic Survival Prediction 🚢

This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## 📊 Dataset

* Source: Kaggle Titanic Dataset

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 📌 Project Steps

* Data Cleaning
* Feature Selection
* Model Building
* Prediction

## 🎯 Result

The model predicts survival based on features like age, gender, and class.
